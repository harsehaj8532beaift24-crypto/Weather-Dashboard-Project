# Weather Dashboard Project

A responsive Weather Dashboard built using HTML, CSS, and JavaScript.

## Features

- Login Authentication using users.json
- Live Weather Search
- 3-Day Weather Forecast
- Dynamic Background
- Logout Functionality
- Responsive Design

## Technologies Used

- HTML
- CSS
- JavaScript
- WeatherAPI

## Setup

1. Visit https://www.weatherapi.com/
2. Create a free account.
3. Copy your API key.
4. Open `js/dashboard.js`
5. Replace:

```javascript
const apiKey = "YOUR_WEATHERAPI_KEY";
```

with your own WeatherAPI key.

6. Run `index.html` using Live Server.
